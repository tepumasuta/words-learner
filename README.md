# words-learner
words-learner - приложение, созданное помочь пользователям в изучении иностранных языков благодаря отлаженной структуре запоминания слов, позволяющее закрепить полученные знания, а также постепенно пополнять свой словарный запас, добавляя новый словосочетания в базу данных приложения.
